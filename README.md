# Typcursus

Met deze typcursus zou iemand zichzelf kunnen leren typen met tien vingers. Allereerst wordt een menu aangeboden. Hier kan een gebruiker kiezen welke rij(en) van het toetsenbord hij/zij wil oefenen. Vervolgens wordt voor die rij steeds een willekeurige letterreeks aangeboden die de cursist met tien letters na moet typen.

## Gebruik

Om de typcursus te starten moet Python zijn geïnstalleerd. Zet vervolgens onderstaande bestanden op één locatie:

- typcursus.py
- menu.py
- toetsenbord.py

Open vervolgens het bestand *typcursus.py* met Python om de cursus te starten.

## Testen

Ten behoeve van het testen, is het mogelijk om in plaats van de zin na te typen, '-' te typen.
